## このディレクトリについて

ここにキャラクターモデルに関するものを置きます。
VRM-1.0形式のVRMファイル default.vrm がデフォルトで読み込まれます。
他のモデルを用いたい時は、default.vrmを差し替えるか、Webインタフェースから差し替えてください。
